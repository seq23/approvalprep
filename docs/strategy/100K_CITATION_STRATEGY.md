# 100K Citation / LLM Surfacing Strategy

The 100K target is a six-month planning goal, not an observed-win claim.

ApprovalPrep tracks four separate numbers:

1. Planned targets: the strategic goal for possible citation-ready surfaces.
2. Generated surfaces: actual pages, atoms, answer assets, indexes, or source maps that exist in the repo.
3. Detected opportunities: query or citation opportunities discovered by intelligence scripts or manual import.
4. Observed wins: surfacing or citation events proven by telemetry or manual review.

Observed wins require proof from Search Console, Bing, manual SERP/LLM checks, citation imports, or paid-source receipts. The repo must not infer wins from generated content.

## Production Requirements

- Build a large atom library across definitions, short answers, checklists, comparisons, product explanations, process steps, and boundaries.
- Generate daily low-risk answer assets and queue regulated content for approval.
- Keep `llms.txt`, `llms-full.txt`, `/answers/index.json`, sitemap, route manifest, atom maps, claim maps, and source maps synchronized.
- Expand the query universe by application situation: credit, rental, income, employment, auto, mortgage, SBA, business funding, and life-admin paperwork.
- Refresh and prune thin or duplicate content.
- Track observed wins separately from targets.

## Current Artifact Foundation

This artifact repairs the foundation by expanding atoms, improving LLM files, generating daily answer assets, protecting KPI honesty, and separating production proof from fixture traces. It does not guarantee 100K observed citations.
