# ApprovalPrep v4.1 Query Intelligence + Growth Maintenance Engine

This engine has two lanes.

1. Content creation intelligence decides whether a query should become a growth page, utility page, noindex page, FAQ, checklist, table/list, product-flow support block, admin-only resource, or blocked safety item.
2. Maintenance intelligence reviews existing pages and decides whether a page needs a table/list, FAQ, source support, internal links, authority/backlinks, refresh, or no action.

Low ranking potential never blocks useful onsite content. It only changes priority, indexing, and authority strategy.

A connector is real only when it calls a live free source, imports a real operator file, crawls an allowed public page, or returns an honest NOT_CONFIGURED, NO_DATA, or SOURCE_ERROR state. Fake rows, sample SERPs, fake rankings, and fake citation wins are forbidden.
